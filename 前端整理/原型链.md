原型链
=====
创建对象有几种方法
* 对象字面量：
````
var O1 = {name:'o1'};
var O11 = new Object({name:'o11'});
````
* 构造函数：
````
var M = function(){
    this.name = 'm';
}
var o2 = new M();
````
* Object.create()
````
var P = {name:'o3'};
var o3 = Object.create(P);
````
